Ce paramètre permet de choisir, dans l'ensemble des utilisateurs enregistrés, un utilisateur qui recevra le message.

Ce réglage est commun à tous les formulaires de contacts utilisés dans ce site.

Si l'adresse que vous voulez notifier est extérieure à Moodle, vous pouvez créer un compte d'utilisateur non actif à cette adresse pour pouvoir trouver cette dernière dans la liste.

Pour désactiver cet envoi, laissez cette liste non "choisie".
